What are the different ways to select an element in the DOM?

To select an element in the DOM, we can use - document.getElementById(id), document.getElementsByClassName(className), document.getElementsByTagName(tagName), and document.querySelector(selector).

What is the difference between innerHTML, innerText, and textContent ?

innerHTML - can read html markup and the text content of the element. innerText - can read content of the element. textContent - ignores all html tags and returns the text content.

What is event delegation in the DOM?

The way to add event listeners to a parent element instead of adding event listeners to each child element.

What is event bubbling in the DOM?

The way to add event listeners to a parent element instead of adding event listeners to each child element.

How do you create, add, and remove elements using JavaScript?

We can create elements using document.createElement(tagName). We can add elements using appendChild() , append(), innerHTML etc, We can remove elements using removeChild(), remove(), innerHTML = "".
